#BLE_Blance_Car
BLE_Blance_Car is a Firmware to control "702 Balance car"

##software features:
* Self balance
* Move forward and back
* Turn left and right
* Obstacle avoidance mode
* Dance mode
* Broadcast
* Rotate head
* Fall detect
